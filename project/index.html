FAQ

よくいただくご質問をまとめました。

Q未経験でも本当に大丈夫ですか？
＋
はい、所属ライバーの約7割が未経験スタートです。
配信機材の選び方 / OBS設定 / トーク / タイトル設計まで、初心者向けカリキュラムをご用意しています。マネージャーが二人三脚で伴走するので、最初の30日で「初配信→収益化」までを目指せます。
Qどのようなゲームを配信すればいいですか？
＋
FPS / バトルロイヤル / RPG / 雑談 / 歌枠など
ご自身が「長く楽しめる」ジャンルを一緒に決めます。
市場分析データを共有し、競合の少ない時間帯・ジャンルへの参入もサポートします。
Q収益化までにどのくらいかかりますか？
＋
所属ライバーの初月収益化達成率は83%。
最短で配信開始から2週間で収益化に到達するケースもあります。
ご本人のペースとプラットフォームに合わせて、無理のないロードマップを引きます。
Qサポートの料金はかかりますか？
＋
所属ライバーのサポート費用は無料です
（収益化後に一定比率の取り分が発生する成果連動型）
事前審査やヒアリングも無料で実施しています。
Q副業や学業と両立できますか？
＋
もちろん可能です。
週5〜10時間の配信から始める方が大半で、学生 / 社会人それぞれの生活に合わせて
配信枠を組んでいます。本業との両立事例も多数。
